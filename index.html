📋 Instruções de deploy
  1. Suba os arquivos no GitHub: index.html, api/data.js, api/save.js
  2. Conecte o repositório à Vercel
  3. No painel da Vercel: Storage → Marketplace → Upstash → Redis → Create & Connect
  4. A Vercel adiciona automaticamente as env vars UPSTASH_REDIS_REST_URL e UPSTASH_REDIS_REST_TOKEN
  5. Faça um redeploy — pronto!
