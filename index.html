Sincronização — Vercel KV
✅ Este sistema usa Upstash Redis (via Vercel Marketplace) como banco de dados. Todas as alterações feitas por qualquer usuário são salvas automaticamente e aparecem nos demais dispositivos em segundos. Nenhuma configuração adicional é necessária após o deploy.

📋 Instruções de deploy
  1. Suba os arquivos no GitHub: index.html, api/data.js, api/save.js, package.json, vercel.json
  2. Conecte o repositório à Vercel (Import Project)
  3. No painel da Vercel: Storage → Marketplace → Upstash → Redis → Create & Connect
  4. A Vercel adiciona automaticamente as env vars UPSTASH_REDIS_REST_URL e UPSTASH_REDIS_REST_TOKEN
  5. Faça um redeploy — pronto! Todos os dispositivos sincronizam a cada 10 segundos
