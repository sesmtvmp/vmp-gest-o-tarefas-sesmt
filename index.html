ℹ️ Configure seu repositório GitHub para que as alterações feitas pelo administrador sejam salvas no data.json e fiquem disponíveis para todos os dispositivos via Vercel.
🔑 Credenciais do GitHub
Crie em: GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic). Permissão necessária: repo (Contents write).
